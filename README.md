# ⚡ Calculadora de Consumo de Energia Residencial 💲

Este projeto foi desenvolvido como parte da *Agenda 05* do curso *Técnico em Desenvolvimento de Sistemas (ETEC)*.

## 🖥️🖥️💰💰 Sobre o Projeto
O objetivo deste script em Python é para que o usuário possa calcular o custo estimado mensal de energia elétrica de um eletrodoméstico, baseando-se na potência do aparelho e no tempo de uso diário.

## 👩‍💻 Tecnologias Utilizadas
* 🐍*Python 3*: Lógica de programação e cálculos.
* 😸*Git & GitHub*: Controle de versão e hospedagem do código.
* 🆚*VS Code*: Ambiente de desenvolvimento.

## 🚀🚀 Como Executar
1. Certifique-se de ter o Python instalado.
2. Execute o arquivo app.py.
3. Insira o nome do aparelho, a potência (em Watts), as horas de uso por dia e o valor do kWh da sua região.
